## Introduction

This page provides a description of SIBiLS (Swiss Institute of Bioinformatics Literature Services). SIBiLS provide different services which are freely accessible via **REST APIs** and are **ready to empower any curation workflow**, built on modern technologies scalable with big data.


## Access the services

<p style="text-align: center" markdown>

</p>
